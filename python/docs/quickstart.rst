Quick Start Guide
=================

Installation
------------

.. code-block:: console

    $ pip install bosing


Build from source
-----------------

#.  Install `.NET 8.0 SDK <https://dotnet.microsoft.com/en-us/download>`_
#.  Install Python 3.8 or later
#.  Clone the repository and install the package

    .. code-block:: console

        $ git clone https://github.com/kahojyun/Bosing.git
        $ cd Bosing
        $ pip install -e .
