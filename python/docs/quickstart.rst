Quick Start Guide
=================

Installation
------------

配置运行环境
^^^^^^^^^^^^

#.  安装 `.NET 8.0 SDK <https://dotnet.microsoft.com/en-us/download>`_
#.  安装 Python 3.8 及以上版本


安装 Client
^^^^^^^^^^^^

#.  安装 ``pulsegen_client``

    .. code-block:: console

        $ git clone https://github.com/kahojyun/pulsegen-client.git
        $ cd pulsegen-client
        $ pip install .
        $ pip install matplotlib
